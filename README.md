## Introduction
Jobfindout is my personal project to pass bootcamp first task and this is website for people to finding and getting a job. This is open source and feel free to use and bought me.
## Prerequiste
 - Node.js
 - MySQL
 - Nodemon
 - Redis
## Installation
### Clone
    $ git clone https://github.com/nukumalik/jobfindout.git
    $ cd jobfindout
    $ npm install
### Create Enviroment Variable
    $ cp .env.example .env
    $ nano .env
### Start Development Server
    $ npm run dev
### Other Depedencies
 - bcryptjs
 - cors
 - express-redis-cache
 - jsonwebtoken
 - passport
 - passport-jwt
 - mysql
 - mysql-redis-cache
 - redis
